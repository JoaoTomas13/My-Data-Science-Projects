SYSTEM REQUIREMENTS

- Java (version 8 or higher)
- JSOUP library (version 1.14.1 or higher)

INSTALLATION INSTRUCTIONS

1. Unzip the project folder.

2. In your IDE, click File -> Open Project and select the project folder.

3. Start each component of the programme in the following order: IndexStorageBarrelRMI -> SearchModule -> ProjectApplication -> Downloader -> RMIClient

4. In your browser, enter the address http://localhost:8080
